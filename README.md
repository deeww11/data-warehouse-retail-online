# Data Warehouse Retail Online

## Deskripsi Proyek
Proyek ini merupakan implementasi Data Warehouse untuk toko retail online yang bertujuan mendukung analisis penjualan dan pengambilan keputusan bisnis.

## Dataset
Data transaksi penjualan retail online.

## Struktur Folder
- Dahsboard
- Master Data
- Transaksi
- Laporan
